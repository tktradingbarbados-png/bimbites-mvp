This is a ready-to-run Next.js starter for BimBites (Barbados delivery MVP). It includes:

- Customer-facing app with search, menu, cart, and WhatsApp checkout
- Admin page at /admin that edits menus in browser localStorage (no redeploy needed)
- Partners pre-populated with realistic menus (Seafood Shack, Mapps, Little Caesars, K's Roti, Yelluh Meat, Milococktail)
- Sample images and favicon

## Quick start

1. Unzip and copy to your repo root.
2. Update your WhatsApp number in `data/partners.js` (whatsappTarget).
3. Install & run:

```bash
npm install
npm run dev
```

4. Open http://localhost:3000 and http://localhost:3000/admin

Push to GitHub and Vercel will auto-deploy. Your domain bimbites.com will continue to work.
